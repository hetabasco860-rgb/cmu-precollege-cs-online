Program Overview

The Pre-College Computer Science Online program provides a structured pathway into foundational and applied computer science. Instruction emphasizes computational thinking, programming practice, and disciplined problem solving through guided learning and milestone-based progress.

The program is designed to support students who are prepared to engage with university-style expectations: consistent preparation, careful attention to correctness, and clear technical communication. Students build skills incrementally across the session sequence and apply those skills in a culminating final project.

Certificate pathway: Students who successfully complete the program and submit the Final Project will receive a Certificate of Completion issued by Pre-College Programs, Department of Computer Science.

Academic Structure

The course spans core AP-aligned foundations and extends into advanced topics beyond the AP curriculum. Across the 20-session sequence, students develop programming proficiency, core conceptual understanding, and applied engineering habits through structured instruction and practice.

  • AP CS A foundations: structured programming, problem decomposition, algorithmic thinking
  • AP CS Principles concepts: data, abstraction, networks, impacts of computing
  • Advanced extensions: selected topics to deepen conceptual and practical readiness
  • Milestones: project-aligned checkpoints that build toward a final submission

Detailed learning objectives and curriculum structure are available on the Course page.

Projects Intro

The Final Project serves as the culminating assessment of the program. Students design and implement a complete software artifact demonstrating programming proficiency, systematized problem solving, and clear technical documentation.

Students are expected to define an appropriate scope, implement reliably, test against representative cases, and document usage and design decisions. The project should be understandable to reviewers based on the submission package alone.

Submission policy: The Final Project is evaluated from a single official submission. Students should plan carefully, validate correctness, and ensure documentation is complete prior to submission.
