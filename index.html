Program Overview

The Pre-College Computer Science Online program is designed for students seeking a rigorous, structured introduction to computer science. Instruction emphasizes computational thinking, programming practice, and foundational theory, with clear expectations and defined milestones.

Projects Intro

A central component of the program is the Final Project, which serves as the culminating assessment. Students design and implement a complete software artifact demonstrating programming proficiency, problem solving, and technical communication.

Submission policy: The Final Project is evaluated from a single official submission. Students are expected to plan carefully, test thoroughly, and submit complete documentation.
