Course Format

The course is designed as a structured sequence of sessions with consistent expectations for preparation, practice, and milestone completion. Each session includes guided instruction and applied exercises aligned with program outcomes.

Learning Objectives

By the end of the program, students are expected to:

  • Demonstrate proficiency in structured programming and algorithmic thinking.
  • Apply core concepts from AP Computer Science A and AP Computer Science Principles.
  • Explain computing concepts with clear reasoning and appropriate technical vocabulary.
  • Develop a complete final project with correct behavior and clear documentation.

Curriculum Outline

The curriculum spans AP foundations and extends into advanced topics selected to match cohort readiness. Emphasis areas include:

  • Programming fundamentals: variables, control flow, functions/methods, debugging, code style
  • Problem solving: decomposition, algorithm selection, correctness reasoning, testing
  • Data & abstraction: data representation, information processing, abstraction practices
  • Advanced extensions: selected topics beyond AP to deepen practical and conceptual understanding

Final Project Alignment

Coursework is designed to prepare students for the Final Project milestone. Project expectations and evaluation criteria are provided on the Projects page.

Certificate Pathway
